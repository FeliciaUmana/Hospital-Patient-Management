Hospital Patient Management Analytics

Project Overview
This project analyzes hospital patient management data to uncover insights into patient admissions, treatment costs, department workload, and healthcare trends.

Using SQL, Python, and Power BI, the project transforms raw hospital records into meaningful insights that help hospital administrators make data-driven decisions.

The analysis focuses on answering key operational questions such as:

* Which hospital departments handle the most patients?
* What are the most common diagnoses?
* Which doctors treat the highest number of patients?
* How much revenue does each department generate?
* What is the average patient length of stay?

Power BI Dashboard
The project includes an interactive hospital analytics dashboard built in **Power BI**.

Dashboard Features
* Total Patients
* Total Hospital Revenue
* Average Treatment Cost
* Average Length of Stay

Visualizations

* Patients by Department
* Monthly Admission Trends
* Diagnosis Distribution
* Doctor Workload
* Patient Gender Distribution

The dashboard enables hospital administrators to explore the data interactively and identify important healthcare trends.

 Key Insights

Some insights discovered during analysis include:

* Certain departments handle significantly more patient cases than others.
* Emergency admissions represent a large portion of total hospital visits.
* Some diagnoses occur far more frequently than others.
* Doctor workloads vary widely across departments.
* Patient admission patterns fluctuate over time.

These insights can help hospitals optimize staff allocation and improve operational efficiency.
